# ComfyUI with EchoMimic V3 Flash

**Полноценная, готовая к работе сборка ComfyUI с предустановленной нейросетью EchoMimic V3 Flash для анимации лиц по аудио.**

Данный репозиторий содержит набор скриптов для **полностью автоматической установки** ComfyUI со всеми необходимыми кастомными нодами и зависимостями на сервере с **Ubuntu 22.04 / 24.04 / 26.04** и **CUDA 13**. Установка занимает 15–30 минут и не требует ручного вмешательства.

**Автор:** [vokilook](https://github.com/vokilook)

---

## 🎯 Что вы получите после установки

- **ComfyUI** (актуальная версия) — графический интерфейс для работы с нейросетями.
- **EchoMimic V3 Flash** — модель для синхронизации губ и мимики персонажа с аудиодорожкой.
- **Video Helper Suite (VHS)** — набор узлов для работы с видео (загрузка, конвертация, сохранение).
- **ComfyUI-Manager** — менеджер для удобной установки и обновления кастомных нод.
- **Все необходимые Python-библиотеки** — автоматическая установка зависимостей для стабильной работы.
- **Python 3.10** — автоматическая компиляция из исходников, если отсутствует в системе.

---

## 📦 Требования к серверу

| Компонент | Минимальные требования |
|-----------|------------------------|
| **Операционная система** | Ubuntu 22.04 / 24.04 / 26.04 |
| **GPU** | NVIDIA с 12+ ГБ VRAM (рекомендуется 24 ГБ, например, RTX A5000) |
| **CUDA** | 13.0 (или совместимая с PyTorch 2.5+) |
| **Драйверы NVIDIA** | 570+ |
| **Дисковое пространство** | ~20 ГБ (с учётом моделей) |
| **Оперативная память** | 32+ ГБ (рекомендуется) |
| **Интернет** | Для скачивания зависимостей и моделей |

> ⚠️ **Важно:** установка выполняется на **чистый сервер** или сервер, где ещё не установлен ComfyUI. Если ComfyUI уже есть — используйте изолированное окружение.
